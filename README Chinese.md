# Salesforce Productivity Analyzer Skill

一个基于因果模型的销售生产力诊断工具，帮助销售管理者系统性地识别绩效差距的根本原因。

---

## 核心模型

```
销售绩效 = 努力数量 × 努力质量 × 外部环境
```

该模型为**乘法结构**：努力数量高但质量低的销售员仍会表现不佳。诊断逻辑按层推进，在确认问题的第一层即停止，避免过度归因。

---

## 适用场景

当你遇到以下问题时，触发此 skill：

- 销售团队整体或个别销售员未达配额
- 需要评估销售员绩效、决定留任/辅导/解雇
- 分析各辖区销售潜力是否被充分开发
- 评估是否需要招聘新销售员
- 改进销售绩效评估体系与 KPI 设计
- 对比不同销售员的效率与投入产出比

**触发关键词示例：** Salesforce 分析、销售生产力、销售绩效诊断、赢单率、每账户收入、销售 ROI、辖区分析、配额达成率、销售漏斗分析

---

## 五步诊断逻辑

按顺序逐层推进，在确认问题的第一层停止。

| 步骤 | 分析维度 | 核心问题 |
|------|----------|----------|
| Step 1 | 结果差异分析 | 差距存在吗？有多大？在哪里？ |
| Step 2 | 努力数量分析 | 销售员的活动量足够吗？ |
| Step 3 | 努力结构分析 | 努力方向是否正确？ |
| Step 4 | 转化质量分析 | 努力是否有效转化为收入？ |
| Step 5 | 外部环境分析 | 是否存在超出销售员控制的制约因素？ |

---

## 核心诊断指标

### 效率指标
- **每账户收入** = 总收入 ÷ 活跃账户数
- **每次拜访收入** = 总收入 ÷ 年拜访次数
- **销售 ROI** = 总收入 ÷（薪酬 + 费用）
- **账户渗透率** = 活跃账户 ÷ 潜在账户

### 辖区公平性指标
- **CMI 调整指数** = 实际区域占比 ÷ CMI 权重占比
  - 高于 1.0 = 超越辖区潜力
  - 低于 0.7 = 显著低于辖区潜力

### 领先指标
- 每季度新开账户数
- 休眠账户再激活率
- 流水线覆盖率（目标 ≥ 3×）

## 数据输入格式

支持以下输入方式：

1. **Excel / CSV 导出** — 读取所有列，映射至模型维度，自动计算可用指标
2. **粘贴表格** — 直接解析，对模糊列名请求澄清
3. **口头描述** — 先收集 Step 1 所需数字，逐层推进
4. **部分数据** — 完成可用步骤，明确标注因数据缺失而跳过的步骤

**建议提供的 Salesforce 字段：**

| 数据类型 | 字段 |
|----------|------|
| 销售员基本信息 | 姓名、辖区、任期 |
| 业绩数据 | 年销售额、历史销售额、配额 |
| 活动数据 | 年拜访次数、活跃账户数 |
| 辖区数据 | 潜在账户数、CMI 指数、人口/面积 |
| 成本数据 | 薪酬、佣金、费用 |
| 产品数据 | 各产品线销售额、毛利率 |

## 输出格式

### 诊断摘要模板

```
诊断摘要
─────────────────────────────────────────────
差距描述：     [结果差距说明]
根本原因层：   [努力数量 / 努力结构 / 转化质量 / 外部环境]
核心发现：     [一句话描述问题]
支撑数据：     [2–3 个关键指标]
建议行动：     [具体、可执行的下一步]
次要发现：     [后续层级发现的其他问题]
```

### 可视化输出
- 销售员绩效对比表（含基准线）
- 漏斗流失分析
- 辖区容量热力图
- CMI 调整后排名

```
用户：我上传了一份新英格兰区域的销售数据表格，
      帮我分析哪些销售员表现不佳，以及是否需要招聘。

  输出人员评估、招聘建议、生产力提升方案和指标改进框架。
```
